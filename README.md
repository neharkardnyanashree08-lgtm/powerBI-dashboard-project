# 📊 Customer Churn & Retention Analytics Dashboard

## 🚀 Project Overview

This project analyzes customer churn behavior using a large dataset (15,000+ records) to identify key factors driving customer attrition and provide actionable insights for improving customer retention. It demonstrates end-to-end data analysis using Excel, SQL, Python, and Power BI.

---

## 🎯 Business Problem

Customer churn directly impacts revenue and growth. This project aims to:

* Identify high-risk customers likely to churn
* Analyze key drivers such as contract type, support calls, and customer activity
* Provide data-driven recommendations to improve retention

---

## 🛠️ Tools & Technologies Used

* **Excel** – Data preprocessing and pivot analysis
* **SQL (MySQL/PostgreSQL)** – Data querying and transformation
* **Python (Pandas)** – Data cleaning and exploratory data analysis
* **Power BI** – Interactive dashboard and business visualization

## 📊 Key Insights

* 📉 Customers with **monthly contracts** show the highest churn rate
* 📞 Higher **support call frequency** is strongly linked to churn
* ⏳ Customers inactive for more than 30 days are more likely to leave
* 💰 Significant revenue loss is associated with churned customers

---

## 🔍 Data Analysis Workflow

1. **Data Collection** – Imported dataset in Excel format
2. **Data Cleaning** – Handled missing values and inconsistencies using Python
3. **Data Transformation** – Structured and queried data using SQL
4. **Exploratory Data Analysis (EDA)** – Identified churn patterns
5. **Visualization** – Built interactive dashboards in Power BI

---

## 📈 Power BI Dashboard Features

* KPI Cards: Total Customers, Churn Rate, Revenue Loss
